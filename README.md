# ☕ Triple Espresso

Projeto de treino desenvolvido com HTML e CSS que apresenta um espaço ideal para estudo, com a possibilidade de realizar reservas para aproveitar os recursos oferecidos.

## 📖 Sobre o projeto

O **Triple Espresso** é uma página web estática que simula um ambiente moderno voltado para estudos e produtividade. O site apresenta informações sobre o espaço e permite ao usuário visualizar a opção de realizar reservas.

Este projeto foi criado como parte do aprendizado inicial em desenvolvimento web, com foco em estruturação com HTML e estilização com CSS.

## 🎯 Objetivo

* Praticar conceitos básicos de HTML
* Aprender estilização com CSS
* Desenvolver a organização de uma página web
* Criar um layout simples e funcional

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

## ▶️ Como visualizar o projeto

Você pode visualizar o projeto de duas formas:

1. Baixando o repositório:

```bash
git clone https://github.com/francielle-dev/web_project_coffeeshop.git 
```

2. Abrindo o arquivo `index.html` no navegador
